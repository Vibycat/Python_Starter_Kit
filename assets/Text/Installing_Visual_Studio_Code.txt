Installing Visual Studio Code 

In this section, we will Install Visual Studio Code on your computer. 

There are multiple ways to do this. 

Option 1: 
Navigate to the src Folder and Double click the Install_VScode_Windows.exe file.
this will install the latest version of  visual studio code on a windows computer. 

Option 2: 
Navigate to the src folder and double click the Install_VScode_Windows.bat file 
this will install the latest version of visual studio code on a windows computer.

Option 3: 
If you are not using a windows computer, Navigate to the Product webpage and install the latest version for your operating system.

Option 4: Skip This step and use your own Text editor and Python I.D.E.
